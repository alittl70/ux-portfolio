Personal Website - Foundational Web Design Project

Course: ITSC 1110 - Introduction to Computer Science
Role: Designer & Developer
Timeline:January - May 2024

This project was my first independently designed and developed website, created as a part of ITSC 1110. The goal was to apply foundaitonal HTML and CSS concepts while creating a clear, welcoming landing page that introduced content in a simple and visually engaging way.

Focus Areas

  • Establishing visual hierarchy using headings and spacing.
  • Designing a simple, approachable landing experience.
  • Using color intentionally to create a cohesive visual identity.
  • Structuring content with semantic HTML elements.

What I learned

While visually simple, this project marked my first experience translating ideas into a digital interface. It introduced me to the importance of hierarchy, spacing, and clarity in guiding users through content.

Overall, this project highlights how my approach to design has evolved. I now prioritize accessability, user-centered decision-making, and iterative refinement-principles that guide my current work in UX/UI design and front-end development.
